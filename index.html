על הקורס בקצרה

תחום האינטרנט צומח בקצב מסחרר והתשתית האינטרנטית הופכת להיות זירת המסחר העיקרית של חברות רבות המרחיבות את יכולותיהן המסחריות וזקוקות לעובדים מקצועיים להקמה, תחזוקה וניהול האתר .
מנהל האתר כיום נדרש להפגין ידע עם ארכיטקטורת וטכנולוגיות Web שונות, הכרות מעמיקה עם מערכות לניהול תוכן, וקידום האתר במנועי החיפוש.
הידע הבסיסי הנדרש לבנייה וניהול אתרי אינטרנט הינו הבנה ויכולות כתיבה בשפת התגיות HTML ועיצוב הגיליונות באמצעות CSS.
קורס זה מתייחס לאספקטים הרלוונטיים למנהלי אתרים ומנהלי תוכן הקשורים לבנייה, תחזוקה וניהול אתרים .
קורס HTML מלמד את הטכנולוגיות הבסיסיות בבניית אתרי אינטרנט HTML ושפת העיצוב CSS המגדירה עיצובים מתקדמים מעבר ליכולות שפת ה- HTML.
משתתפי הקורס יחשפו לחידושים ב- HTML5 וב- CSS3 ובנוסף ילמדו כיצד לעבוד עם מערכת ניהול התוכן הפופולארית WordPress.
